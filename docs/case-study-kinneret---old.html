Case Study: Kinneret - OLD

About

Lake Kinneret is a freshwater lake (32 ÂºN, 35 ÂºE) situated in northern Israel, and is the lowest freshwater lake in the world (209m below Mediterranean Sea Level). The lake has a surface area of 173 km2, and is about 49 m deep. The model is set-up to simulate the hydrological domain of Lake Kinneret for 2 years, from 1997-01-01 to 1999-01-01 (&time), with water balance and heat fluxes hypothetically calculated based on the lake configuration and input data. Water quality functionality (AED2) is disabled for the simulation (&wq_setup).

The Kinneret97 model simulation above has the ecological model aed2 enabled. It has been pre-configured to run the customizable AED modules. These are separate models, but have been set up to run side by side. They are configured using the text file aed2.nml.
